Данный проект связан с сайтом, на котором собрана финансовая и банковская информация. Данная информация берется из программного интерфейса приложения (api) Национального банка Республики Беларусь и коммерческих банков.
    Основными особенностями проекта являются:
  • Работа с банковскими api
  • Графики на основе сторонних библиотек
  • Онлайн конвертер валют
    Общие требования к проекту:
  • Наличие презентации к проекту
  • Размещение кода на github pages
  • Заранее утвержденная структура проекта
  • Написание кода с использованием TSLint и подготовленным конфигом для него
  • Проект без тестов
